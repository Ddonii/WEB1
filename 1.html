HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.

Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. HTML elements are delineated by tags, written using angle brackets.
